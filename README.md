## Welcome to Oxidize

This project is an Operating System written in Rust. The proposal of this system is build a modulable and functional OS built from scratch.

## Getting Started

Clone the project:

```
git clone git@github.com:vgsantoniazzi/oxidize.git
```

## Usage

##### Compile

```
make build
```

##### Run

```
make all
```
